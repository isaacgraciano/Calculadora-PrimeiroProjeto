# 🧮 Calculadora Python

Uma calculadora simples e direta, feita em Python puro — sem interface gráfica, só a essência do código.

## 📋 Funcionalidades
- ➕ Adição
- ➖ Subtração
- ✖️ Multiplicação
- ➗ Divisão
- 🧹 Limpar resultado
- 🔄 Operações contínuas

## 🎯 Objetivo
Projeto criado pra treinar lógica de programação, estrutura de funções e manipulação de entrada do usuário no terminal. Simples, funcional e direto ao ponto.

## 🛠️ Tecnologia
- Python 3.x
- 0 dependências externas — só Python puro
